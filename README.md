# AI-Powered Object Detection & Tracking System

A Streamlit-based computer vision application that performs real-time object detection and tracking using YOLOv8 and ByteTrack.

## Features

- Image Detection
- Video Detection
- Object Tracking
- Confidence Threshold Control
- Detection Metrics
- Download Processed Video

## Tech Stack

- Python
- OpenCV
- Streamlit
- YOLOv8
- ByteTrack

## Run

pip install -r requirements.txt

python -m streamlit run streamlit_app.py
